As Linguagens de programação

As linguagens de programação foram criadas para solucionar qualquer tipo de problema na área tecnológica computacional. Acompanhando todas as tendências da tecnologia, elas estão sempre em constante evolução, buscando oferecer as mais modernas ferramentas para as mais complexas tarefas.

Alguns Conceitos

É uma linguagem formal que funciona por meio de uma série de instruções, símbolos, palavras-chave, regras semânticas e sintáticas. A linguagem de programação permite que um programador crie programas a partir de um conjunto de ordens, ações consecutivas, dados e algoritmos. Cada linguagem possui suas particularidades e, no conteúdo de hoje, conheceremos algumas delas e suas principais aplicações.

AS 3 Linguagens mais usadas

JavaScript

JavaScript é uma linguagem de programação interpretada estrutura,
de Script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.

Java

Java é uma linguagem de programação orientada a objectos desenvolvida na década de 90 por uma equipe de programadores chefiada por James Gosling, na empresa Sun Microsystems, que em 2008 foi adquirido pela empresa Oracle Corporation.

Python

Python é uma linguagem de programação de alto nível, interpretada de script, imperativa, orientada a objectos, funcional, de tipagem dinâmica e forte. É Open-source de propósito geral usado bastante em data science e machine learning.

Outras Linguagens usadas

Go

Go é uma linguagem de programação criada pelo Google e lançada em código livre em novembro de 2009. É uma linguagem compilada e focada em produtividade e programação concorrente, baseada em trabalhos feitos no sistema operacional.

Kotlin

Kotlin é uma linguagem de programação multiplataforma, orientada a objectos e funcional, concisa e estaticamente tipada, desenvolvida pela JetBrains em 2011, que compila para a máquina virtual Java e que também pode ser traduzida para a linguagem.

C

C é uma linguagem de programação compilada de propósito geral, estruturada, imperativa, procedural, padronizada pela organização internacional para Padronização, criada em 1972 por Dennis Ritchie na empresa AT&T Bell Labs.
